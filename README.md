## Introduction  
获取当前IPV6并更新到dnspod的AAAA记录.  
使用腾讯云API。  
secretId与secretKey从.env文件中获取  
.env文件示例：  
```
secretId=
secretKey=
```  

## Install
Node.js写的， npm install即可  
然后windows上可以双击run.bat。  
别的平台自己写别的run脚本  

## Background  
家用宽带的IPV6的公网是通的！！再也不用花生壳了！！
NAS / P2P / 文件传输 / 音视频可以搞起来了！！
